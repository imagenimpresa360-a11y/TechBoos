Mapa de Navegación "Cero-Fricción"

El operador registrará todas las acciones comerciales en un único panel de control central (SPA), bloqueando la necesidad de cambiar páginas y perder la transacción de caja.

[ PANTALLA PRINCIPAL: DASHBOARD INTERACTIVO Y CAJA FUERTE ]
  │
  ├─► [Panel Izquierdo] REGISTRAR SALIDA DE DINERO (Caja Chica)
  │    ├─ Modal Flotante: Categoría (Aseo, Sueldos, etc.)
  │    └─ Ingreso de Monto ─► Botón 'Registrar' ─► (Se refleja en Finanzas)
  │
  ├─► [Barra Superior Central] BUSCADOR UNIVERSAL EXPRÉS
  │    └─ Campo: DNI / RUT celular del Cliente
  │         │
  │         ├─► [NO Existe el Cliente] 
  │         │    └─ Alerta Flotante: ALTA EXPRÉS ─► Nombre y Fono ─► PAGO ─► Membresía Activa
  │         │
  │         └─► [SÍ Existe el Cliente]
  │              └─ Tarjeta Lateral Flotante: HISTORIAL DE COMPRAS
  │                   └─ Estado actual ─► Botón "RENOVAR PLAN" ─► PAGO ─► Membresía Activa
  │
  └─► [Sección Privada] ANÁLISIS DE NEGOCIO Y DIRECCIÓN
